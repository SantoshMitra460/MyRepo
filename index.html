A page consisting purely of text looks boring and unattractive. To make sure you page appeals to the user, you need to be able to style it so that it looks beautiful. There are two ways to do that. One is by using an inline CSS. Another one is by linking an external stylesheet. The latter is preferred in most cases as it more convenient, but you can also use inline CSS if you want and don't want to harass yourself with the trouble of linking external style sheets. To use inline CSS, just use the style tags and then write your CSS code within those tags.
